# 🔐 Password Strength Analyzer

A web-based application that evaluates password strength in real-time.

## 🚀 Features
- Password strength detection (Weak / Medium / Strong)
- Entropy calculation
- Crack time estimation
- Password reuse detection
- Smart suggestions
- Show/Hide password option

## 🛠️ Tech Stack
- Python (Flask)
- HTML, CSS, JavaScript
- bcrypt

## 📦 How to Run

1. Install requirements:
pip install -r requirements.txt

2. Run app:
python app.py

3. Open browser:
http://127.0.0.1:5000/

## 📌 Note
This app does not store passwords. It only analyzes them.
